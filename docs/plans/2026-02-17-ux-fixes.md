## Task 1: Fix subagent stuck — use `exit` event instead of `close`

The `close` event fires when all stdio streams close, not when the process exits. If `pi` spawns child processes that inherit stdout, `close` never fires. Switch to `exit` event so the promise resolves when the process terminates.

**Files:**
- Modify: `extensions/subagent/index.ts` (~line 378-385)
- Test: `tests/extension/subagent/subagent-exit-handling.test.ts`

**Step 1: Write the failing test**

Create `tests/extension/subagent/subagent-exit-handling.test.ts`:

```typescript
import { describe, expect, it, vi } from "vitest";
import { EventEmitter } from "node:events";
import type { Readable } from "node:stream";

// We can't easily unit-test the full runSingleSubagent since it spawns a real process.
// Instead, test the behavior we want: that `exit` event resolves the promise, not `close`.
// This is a behavioral contract test using a mock process.

describe("subagent exit handling", () => {
  it("should resolve on exit event even if close has not fired", async () => {
    // This test validates the fix: we listen for 'exit', not just 'close'
    const proc = new EventEmitter() as any;
    proc.stdout = new EventEmitter() as Readable;
    proc.stderr = new EventEmitter() as Readable;
    proc.killed = false;
    proc.kill = vi.fn();

    let resolved = false;
    const promise = new Promise<number>((resolve) => {
      proc.on("exit", (code: number | null) => {
        resolved = true;
        resolve(code ?? 0);
      });
      proc.on("error", () => resolve(1));
    });

    // Process exits but close hasn't fired (stdio still open)
    proc.emit("exit", 0);
    const code = await promise;

    expect(resolved).toBe(true);
    expect(code).toBe(0);
  });
});
```

**Step 2: Run test to verify it passes (this is a design test, it should pass immediately)**

Run: `npx vitest run tests/extension/subagent/subagent-exit-handling.test.ts`

**Step 3: Modify `runSingleSubagent` to use `exit` instead of `close`**

In `extensions/subagent/index.ts`, find the `proc.on("close", ...)` block (~line 378):

Replace:
```typescript
      proc.on("close", (code) => {
        if (buffer.trim()) processLine(buffer);
        resolve(code ?? 0);
      });
```

With:
```typescript
      proc.on("exit", (code) => {
        // Use 'exit' not 'close': close waits for all stdio streams to end,
        // which hangs if pi's child processes inherit stdout.
        // Drain any remaining buffered data from stdout before resolving.
        proc.stdout.once("end", () => {
          if (buffer.trim()) processLine(buffer);
          resolve(code ?? 0);
        });
        // Safety: if stdout doesn't end within 5s after exit, resolve anyway
        setTimeout(() => {
          if (buffer.trim()) processLine(buffer);
          resolve(code ?? 0);
        }, 5000);
      });
```

/**
 * Given a source file path, return candidate test file paths to check for.
 */
export function findCorrespondingTestFile(filePath: string): string[] {
  const ext = path.extname(filePath);
  const base = filePath.slice(0, -ext.length);
  const dir = path.dirname(filePath);
  const name = path.basename(filePath, ext);

  const candidates: string[] = [];

  // Same directory: foo.test.ts, foo.spec.ts
  candidates.push(`${base}.test${ext}`, `${base}.spec${ext}`);

  // __tests__ directory: src/utils/__tests__/foo.test.ts
  candidates.push(path.join(dir, "__tests__", `${name}.test${ext}`));
  candidates.push(path.join(dir, "__tests__", `${name}.spec${ext}`));

  // tests/ mirror: src/utils/foo.ts → tests/utils/foo.test.ts
  if (filePath.startsWith("src/")) {
    const relative = filePath.slice(4); // remove "src/"
    const relBase = relative.slice(0, -ext.length);
    candidates.push(`tests/${relBase}.test${ext}`, `tests/${relBase}.spec${ext}`);
  }

  return candidates;
}
```

Add `import * as path from "node:path";` at the top if not already there.

function persistState(state: FullState): void {
  const stateDir = path.join(process.cwd(), ".pi");
  fs.mkdirSync(stateDir, { recursive: true });
  const statePath = path.join(stateDir, "superpowers-state.json");
  fs.writeFileSync(statePath, JSON.stringify(state, null, 2));
}
```

Modify `reconstructState()`:

```typescript
function reconstructState(): FullState | null {
  const statePath = path.join(process.cwd(), ".pi", "superpowers-state.json");
  if (fs.existsSync(statePath)) {
    try {
      const raw = fs.readFileSync(statePath, "utf-8");
      return JSON.parse(raw);
    } catch {
      return null;
    }
  }
  return null;
}
```

Keep session entry persistence as a fallback write (for backward compat), but read from file first.
